# Usage
```bash
# 任意のフォルダに
git clone 
# cloneしたフォルダへ移動
cd try_sinatra
# gemのセッティング
bundle install
# sinatraサーバーの立ち上げ
bundle exec ruby app.rb
# http://localhost:4567/memo へアクセス
```
あとはメモとしてお使いください。
